# Mobile-Sales-Dashboard
Motorola Sales Dashboard | Power BI Project

A professional and interactive Power BI dashboard developed to analyze and visualize Motorola mobile sales data. This project focuses on transforming raw sales data into meaningful business insights using dynamic visualizations, KPIs, maps, filters, and customer analytics.

The dashboard helps track sales performance, customer behavior, transaction trends, and product performance across different cities and time periods.

Project Overview

The dashboard provides a complete overview of:

Total Sales Revenue
Quantity Sold
Number of Transactions
Average Sales Value
Customer Ratings
Sales by City
Sales by Mobile Model
Payment Method Analysis
Monthly & Daily Sales Trends
